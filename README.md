# CSCI_Lab2
Basic C++ Applications

This lab contains three beginner-level C++ applications focused on practicing user input, calculations, conditional statements, and basic program logic.

The first application is an appliance store calculator that allows users to select an item, apply a discount, calculate sales tax, and display the final purchase price.

The second application is a Human Resources job grading system that takes an employee's name and salary, then determines their job grade level based on salary ranges while handling invalid salary amounts.

The third application is a basic calculator that takes two numbers and performs arithmetic operations including addition, subtraction, multiplication, and division based on user input.

Concepts practiced:

User input and output
Variables and data types
Arithmetic operations
Conditional statements (if/else)
Input validation
Basic problem solving
